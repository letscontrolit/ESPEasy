#Adafruit TCS34725 Color Sensor Driver #

This driver is for the Adafruit TCS34725 Breakout.
    ------> http://www.adafruit.com/products/1334

## About this Driver ##

These modules use I2C to communicate, 2 pins are required to  
interface
