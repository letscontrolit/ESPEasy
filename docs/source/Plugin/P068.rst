Description
-----------

Plugin to use the Sensirion Humidity and Temperature Sensor SHT30 / SHT31 / SHT35.

SHT3x Specifications:
 * Temperature (-40 °C to +125 °C)
 * Relative humidity (0 - 100 %RH)

----------
